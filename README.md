# Hypothesis-testing
This quantitative analysis using hypothesis testisting establishes whether the probability of default of a new IPOed firm (small cap) is significantly higher than a well established firm without considering observable noise in variance of assets. The 'New Small-Cap Issuer' profile is calibrated to real 2025-2026 market medians (leverage, coverage, spreads, historical default rates) for B/B- rated small-cap issuers, all cited on the Data Inputs tab. The comparative firm is Microsoft.
